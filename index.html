Projects

Top 15 Best Sellers

The top 15 best sellers is an app that was built using React and pulls information from the New York Times API. As they update their best sellers this updates my app also and will keep the information upto date. The information this shows is the author of the book, the duration of weeks on the list, the links of where you can purchase the books and also a description of the story.

Find (events)

Find (events) was built with HTML, Sass and Vanilla JS, it works with the Ticketmaster API and pulls information about events, the dates, and venue. The individual would enter the city they are visiting, the start and end dates and click on find events, this will then pull up all the events happening between those dates for that city.

Find (events) was built by myself, Aaron Rodriguez and Tiffany Fust.

Bootcamp Diaries

Bootcamp Diaries was built with React and Firebase, this is an app that allows the individual to write and journal about whatever they feel, it also has a feature where you can add a prompt to the list of prompts we have or you can get a prompt for what to write about when suffering from those pesky writer blocks.

Bootcamp Diaries was built by myself, Corey Sheldrick, Tiffany Fust and Michael Creighton.
